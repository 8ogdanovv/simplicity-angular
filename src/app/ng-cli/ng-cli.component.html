Angular CLI is a pretty cool tool, feature of Angular with those we'll start on our tour.

So...Let's Go ! ! !

If You are reading this text it's more likely that You realize basics of HTML, CSS, JS, and already installed Node JS (that includes Node Package Manager) and some kind of IDE (like VSCode).

You can use the Angular CLI to run commands in your terminal for generating, building, testing, and deploying Angular applications.
To install the Angular CLI, run the following command in your terminal:

npm install -g @angular/cli

It'll install Angular CLI globaly (-g) with help of Node Package Manager. It may not works w\o

sudo
in some cases according to the dependencies of Debian-cored OS and Bash.

If You allready installed Angular CLI You can check its' version by command:
ng v

Angular CLI can help You create your first single-page application.
Simply just choose in your command-line a directory wich will be main-containing (root) relatively to your project.
And than type command
ng new
& e.g.
simple-angular
< as your project title name in your terminal window.
In this case Angular CLI automaticaly creates new application in chosen root directory in folder named as you've type in order to your project title.

While the project is building CLI will ask You about using Routing and CSS preprocessors - so You can config those options personaly.
According to this Simple-Angular tour we've turned router ON as default at the start and turn no one of complex CSS polyfills on.

You can choose directory with command
cd
& e.g.
simple-angular
(or exactly your project directory title name).
and than run your application with command
ng serve -o
at localhost (as default on port :400 http://localhost:4200/) & open it in a new window what flag
-o
means (
--open
is the same).
Another typicaly commands are:
ng generte component
& e.g.
ng-cli
- it'll make new angular component "ng-cli", and its' directory named "/ng-cli" incuding .componet.html, .component.css, .component.ts & .component.spec.ts files.
You can also use shorter syntax to generate the same component:
ng g c ng-cli
, where "ng-cli" is the name of your new Angular component. It'll also works.


Another Angular CLI commands You will see if You'll type
ng help
