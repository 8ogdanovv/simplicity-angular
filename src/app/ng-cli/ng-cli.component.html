If You are reading this text it's more likely that You realize basics of HTML, CSS, JS, and already installed Node JS (that includes Node Package Manager) and some kind of IDE (like VSCode).

You can use the Angular CLI to run commands in your terminal for generating, building, testing, and deploying Angular applications.

To install the Angular CLI, run the following command in your terminal:

npm install -g @angular/cli

It'll install Angular CLI globaly (-g) whis help of Node Package Manager. It may not works w\o

sudo
in some cases according to the features of Debian-cored OS and Bash.

If You allready installed Angular CLI You may check its' version by command:
ng v

Angular CLI can help You create your first single-page application.
Simply just choose in your command-line a directory wich will be main-containing (root) relatively to your project.
And than type command
ng new
& e.g.
simple-angular
< as your project title name in your terminal window.
cd
& e.g.
simple-angular
and than
ng serve -o
that is requred to run localhost (& open it in a new window what flag -o means (--open also))
