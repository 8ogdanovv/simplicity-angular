ng-router {{ title }}

The application already uses the Angular Router to navigate internal path routing.
In ng-cli chapter You have already know how to generate component.
Now we've going to describe ng-router and its dependencies.

First of all we should go to app.module.ts file and add a route path for our component.
It makes this path available for app-routing.module.ts.

Next step You can use your internal links. For this goal You may just add

routerLink="./cli" routerLinkActive="active"

to anchor link or button element.
In this case new link route-path will get us to ./cli internal link.

Also for more informaton You can follow this link https://angular.io/guide/router.
